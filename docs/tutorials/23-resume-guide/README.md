# 第 23 章: 简历优化与自我介绍模板

> **核心目标**：把你做的 Agent 项目包装成面试官一眼就能看到亮点的简历条目。

## STAR 法则在 Agent 项目中的应用

| 要素 | 描述 | 示例 |
|------|------|------|
| **Situation** | 背景和问题 | 公司文档查询效率低，平均耗时 15 分钟/次 |
| **Task** | 你的任务目标 | 构建一个智能文档问答 Agent |
| **Action** | 你做了什么 | 设计 RAG + ReAct 架构，集成 Memory 系统 |
| **Result** | 量化结果 | 查询时间缩短到 30 秒，准确率 92% |

## 自我介绍的"三句话"结构

```
第一句：我是谁 + 技术栈 + 核心方向
"我是 XXX，有 X 年全栈经验，近半年专注 AI Agent 开发。"

第二句：最拿得出手的项目
"最近做了一个文档智能助手 Agent，用 RAG + ReAct 实现了知识库问答。"

第三句：区别于其他人的亮点
"我比较注重工程落地——给项目配了完整的测试和评估体系，也做了成本优化。"
```
